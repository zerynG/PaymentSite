# Руководство по использованию темной темы

## Обзор

Создана универсальная система темной темы, которая автоматически применяется ко всему сайту без необходимости изменять каждый HTML шаблон отдельно.

## Структура файлов

- `static/css/dark-theme.css` - Основной CSS файл с полной поддержкой темной темы
- `static/js/theme-toggle.js` - JavaScript для автоматического управления темой
- `templates/base.html` - Единый базовый шаблон с подключенной темой

## Как использовать

### Вариант 1: Наследование от общего базового шаблона (рекомендуется)

В ваших шаблонах просто наследуйтесь от общего базового шаблона:

```django
{% extends 'base.html' %}

{% block title %}Название страницы{% endblock %}

{% block content %}
    <!-- Ваш контент здесь -->
{% endblock %}
```

Темная тема будет автоматически подключена и работать!

### Вариант 2: Подключение вручную в существующих шаблонах

Если вы не хотите менять структуру наследования, просто добавьте в `<head>` вашего шаблона:

```django
{% load static %}
<link href="{% static 'css/dark-theme.css' %}" rel="stylesheet">
```

И перед закрывающим тегом `</body>`:

```django
{% load static %}
<script src="{% static 'js/theme-toggle.js' %}"></script>
```

Также убедитесь, что у тега `<html>` есть атрибут `data-theme`:

```html
<html lang="ru" data-theme="light">
```

## Переключение темы

Тема автоматически инициализируется при загрузке страницы. Для переключения темы добавьте кнопку:

```html
<button id="theme-toggle-btn" class="btn btn-outline-secondary">
    <i class="fas fa-moon"></i>
    <span id="theme-toggle-text">Темная тема</span>
</button>
```

Или используйте атрибут `data-theme-toggle`:

```html
<button data-theme-toggle class="btn btn-outline-secondary">
    <i class="fas fa-moon"></i>
    <span data-theme-text>Темная тема</span>
</button>
```

## Особенности

- ✅ Автоматическое определение системной темы (светлая/темная)
- ✅ Сохранение выбранной темы в localStorage
- ✅ Поддержка всех элементов Bootstrap
- ✅ Кастомные стили для таблиц, форм, модальных окон и т.д.
- ✅ Плавное переключение без мерцания
- ✅ Поддержка кастомных элементов интерфейса

## CSS переменные

Тема использует CSS переменные для легкой кастомизации:

```css
--bg-primary: #1a1a1a;
--bg-secondary: #2d2d2d;
--bg-tertiary: #3a3a3a;
--text-primary: #e0e0e0;
--text-secondary: #b0b0b0;
--border-color: #404040;
--link-color: #5d9cec;
```

Вы можете переопределить эти переменные в своих стилях для кастомизации темы.
